# Accompagnement Ménopause — Centre Body Reset

La ménopause n'est pas une fatalité — c'est une transition qui s'accompagne.

## Symptômes traités

- Bouffées de chaleur et sueurs nocturnes
- Prise de poids inexpliquée (redistribution abdominale des graisses)
- Troubles du sommeil et fatigue chronique
- Sautes d'humeur, irritabilité
- Sécheresse cutanée, perte de tonicité
- Fuites urinaires d'effort (plancher pelvien affaibli)

## Notre approche

Charlotte établit un bilan complet : symptômes, antécédents, mode de vie, alimentation, sommeil. Le protocole est construit sur mesure : nutrition spéciale ménopause (phytoestrogènes, oméga-3, calcium, vitamine D) + technologies les plus pertinentes.

## Technologies utilisées

- **LED Infrarouge** : réduit les bouffées de chaleur, améliore le sommeil, anti-inflammatoire
- **Cryolipolyse** : cible les amas adipeux abdominaux liés à la redistribution hormonale
- **Adipologie** : raffermit la silhouette modifiée
- **Pressodynamie** : rétention d'eau, jambes lourdes
- **Chaise EMS** : renforcement du plancher pelvien, fuites urinaires
